animation-direction动画循环播放的方向

.bg{width:200px;height:200px;background:green;margin:0 auto; margin-top: 150px;}
.bg:hover{ -webkit-animation: rainbow 3s alternate 1s infinite;}
设定动画执行的方向，其值可以是normal（正常顺序播放）或alternate（反向播放）
@-webkit-keyframes rainbow {  0% { background: green; }  50% { background: blue; }  100% { background:red; }}
