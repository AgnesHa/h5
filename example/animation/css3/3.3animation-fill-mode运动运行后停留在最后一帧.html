animation-fill-mode运动运行后停留在最后一帧

animation:2s rainbow backwards

.bg{width:200px;height:200px;background:green;margin:0 auto;}
.bg:hover{ -webkit-animation:2s rainbow backwards; }
animation-fill-mode : none | forwards | backwards ;
（1）none：默认值，回到动画没开始时的状态
（2）backwards：让动画回到第一帧的状态
（3）forwards：让动画停留在最后一种状态
@-webkit-keyframes rainbow {  0% { background: green; }  50% { background: blue; }  100% { background:red; }}

animation:2s rainbow forwards
