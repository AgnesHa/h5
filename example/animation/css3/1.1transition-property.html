动画变换属性：transition-property

动画变换的属性：transition-property 例如width,height,color等
transition-property是用来指定当元素其中一个属性改变时执行transition效果，其主要有以下几个值：
none(没有属性改变)；
all（所有属性改变）这个也是其默认值；
indent（元素属性名）；
当其值为none时，transition不执行，
当指定为all时，则元素产生任何属性值变化时都将执行transition效果，ident是可以指定元素的某一个属性值。

一个属性变换 transition: width 1s

.bg1{width:200px;height:200px;background:green;margin:0 auto;-webkit-transition: width 1s;}
.bg1:hover{ width: 400px;}

多个属性变换（写法一） transition: height 1s,width 1s

.bg2{width:200px;height:200px;background:green;margin:0 auto;-webkit-transition: height 1s,width 1s;}
.bg2:hover{ height:300px;width:300px;}

多个属性变换（写法二） transition: all 1s

.bg3{width:200px;height:200px;background:green;margin:0 auto;-webkit-transition: all 1s;}
.bg3:hover{ height: 300px;width:300px;}

多个属性变换（写法三） transition: 1s

.bg4{width:200px;height:200px;background:green;margin:0 auto;-webkit-transition: 1s;}
.bg4:hover{ height: 300px;width:300px;}
